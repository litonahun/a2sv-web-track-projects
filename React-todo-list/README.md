# React + TypeScript Todo List App

A simple Todo List application built with **React** and **TypeScript**.  
Users can **add, edit, delete, and mark tasks as completed**. Todos are saved in **localStorage** so they persist across browser refreshes.

## Features

- Add a new todo
- Edit existing todos
- Delete todos
- Mark todos as completed
- Responsive design with Bootstrap
- LocalStorage persistence

## How to Run

1. Clone the repository:

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev

```

4. Open the app in your browser at http://localhost:5173.
